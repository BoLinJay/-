# input 标签  label

```html	
<label for="username">用户名：</label>
<input type="text" id="username"/> //点击“用户名”文字 也可实现聚焦
```

> 点击“用户名”文字 也可实现聚焦

# button

```html
<button type="submit">提交</button>  
```

> type属性： submit 具有刷新页面效果
